# Stay Inspired

> When the sound feels right, everything else falls into place. Stay inspired, stay confident, stay you.

A stylish front-end project built around premium audio branding and a modern product showcase. It includes dedicated pages for the home experience, about section, and product listing, along with organized assets and reusable styling.

## Project Structure

```bash
.
├── Images/
├── Pages/
│   ├── About.html
│   └── Products.html
├── Styles/
│   ├── about.css
│   ├── layout.css
│   ├── main.css
│   └── products.css
└── index.html
```

## Features

- Clean landing page with strong visual branding.
- Dedicated About and Products pages.
- Separate CSS files for layout, main styling, about page, and product page.
- Organized image assets for hero sections, product cards, icons, and background media.
- Ready for a responsive earpods or audio product website.

## Assets

The project includes multiple visuals such as:

- Hero and background images.
- Product images for different earpods styles.
- Icons and support graphics.
- A background video asset for a richer homepage experience.

## Pages

- `index.html` - Main landing page.
- `Pages/About.html` - About the brand or product story.
- `Pages/Products.html` - Product showcase page.

## Styles

- `Styles/main.css` - Global styles.
- `Styles/layout.css` - Layout and structure.
- `Styles/about.css` - About page styling.
- `Styles/products.css` - Products page styling.

## Getting Started

1. Clone the repository.

   ```bash
   git clone https://github.com/yourusername/your-repo-name.git
   ```

2. Open the project folder in your editor.

3. Launch `index.html` in your browser.

## Customization

You can easily adapt this project by:

- Replacing product names and images.
- Updating the branding text and slogan.
- Adjusting colors, fonts, and spacing in the CSS files.
- Adding more product sections or interactive elements.
